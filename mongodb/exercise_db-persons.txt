4. Mostrar solo los nombres y apellidos de aquellas personas que NO tienen como deporte: futbol o son de Rusia.

    db.personas.find({ $or: [{ deportes: { $ne: 'futbol' }}, {pais: 'Russia'} ]}, {nombre: 1, apellido: 1, _id: 0 } )  
        se realiza la búsqueda or ( { $or: [ {},{} ] }, {project} )
            en el project se especifica con 1 lo que quiero que aparezca en la búsqueda y con 0 lo que no.

5. Mostrar las segundas 5 personas que sean de Argentina o Colombia, y el deporte sea Voley, y no hablen Portugues.
